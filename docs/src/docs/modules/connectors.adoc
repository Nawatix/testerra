= Connectors

include::../properties/property-attributes.adoc[]

While using the framework some external connectors could be useful.
The maintainers provide some of them in seperate repositories.
Please take a look at the following list if there is a connetcor mapping your need.

[cols="2,^1,4a",options="header"]
|===
| Name | Documentation | Description
// | Ai4test Connector | https://github.com/telekom/testerra-ai4test-connector/[GitHub] | Experimental AI augmented testing for Testerra
| Appium Connector | https://github.com/telekom/testerra-appium-connector/[GitHub] | Uses the open source standard Appium to run web tests based on Testerra on mobile devices.
| Azure DevOps Connector | https://github.com/telekom/testerra-azure-devops-connector/[GitHub] | Automatic test result synchronization for Microsoft AzureDevOps platform.
| Cucumber Connector | https://github.com/telekom/testerra-cucumber-connector/[GitHub] | Provides the opportunity to use Cucumber and Gherkin to specify `.feature` files and combine it with the advantages of Testerra.
| HPQC Connector | https://github.com/telekom/testerra-hpqc-connector/[GitHub] | Automatic test result synchronization to HP Application Lifecycle Management, former called HP QualityCenter.
| Selenoid Connector | https://github.com/telekom/testerra-selenoid-connector/[GitHub] | Using a Selenium Grid based on https://github.com/aerokube/selenoid[Selenoid] this module provides access to videos and VNC streams.
| TeamCity Connector | https://github.com/telekom/testerra-teamcity-connector/[GitHub] | A simple notification service for Jetbrains TeamCity.
| Xray Connector | https://github.com/telekom/testerra-xray-connector/[GitHub] | Automatic test result synchronization Jira XRay.
|===
